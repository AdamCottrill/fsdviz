The Image model in the FSDB Admin provides an interface for site administrators to upload images that can be referenced and embedded in other places in the FSDB application (eg. About page). In order to upload and use an image, follow these steps:

  1. Click on the 'Add Image' button on the right side of this page
  2. Provide a meaningful title for the image and upload the file.
  3. Click save.

To reference an image in somewhere in FSDB, simply paste this snippet into the text box of the edit form for that object: 

            
                <img src="..." class="ui fluid image"  alt="...">

Provide the url listed in the table below as the source (src) and an alternative text description (alt). Make sure that the alternative text provides sufficient detailed for assistive technologies.

The default size of the image is the full width of the page. To control the width of the image, insert the following code into your snippet (and fill in your desired relative width including the % sign): 

             
                 style="width:...%"
